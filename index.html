Line Follower

Language: C and MIT App Inventor to build the APP

Material: Arduino Uno (ATMEGA 328p), LCD, HC-06 bluetooth, infrared sensors, DC motor ...

Summary:

  • The main purpose was to develop and implement a line follower robot that would have to follow the line as fast as possible.
  • For that we projected a microcontroller based system with a user friendly mobile application that can show everything about the robot, active sensors, battery level, track number, track timer, change velocity of the robot, start and stop...etc
  • This robot has 2 modes, automatic mode that follow the line automatically or manual mode, in this mode the robot is like a RC car, we can control with a APP.
  • With this project we covered many topics of AVR, like interruptions, USART protocol, PWM, A/D converter to see the battery status and EPROM to record the track.

For more info please check on Github

Hydroponic Greenhouse

Language: C, SQL, HTML, CSS and JavaScript

Summary:

The project consists on the development of an information system, with moderate complexity, which includes 3 main layers:

  1. Nodelayer: A multi-threaded software application for sensing/actuation using and embedded platform, namely a Raspberry Pi 3 in C
  2. Server/cloudlayer: A relational database for persistent storage of information. The RDBMS to be used is the PostgreSQL
  3. Application layer: A web interface, to present the information in a tabular and graphical form, when accepting SQL commands of an end user. The preferred languages to be used are HTML, CSS and PHP for the web interface

Weight Measurement with FSR

Material: Insole with FSR and LabView to see the pressure in each sensor

Summary:

  • The main goal was to project and implement a weight measurement system based on FSR (Force Sensing Resistor) technology.
  • For that, we had to develop an electronic circuit for signal conditioning and a virtual instrument to acquire the signals from a data acquisition board.
  • Afterwards we characterized both static and dynamic behavior of the FSR sensors to calibrate them and to have a reliable representation of the applied pressure.
